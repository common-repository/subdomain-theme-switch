== Description ==
This plugin switch if your site is being on subdomain  to an different, selectable theme..

== Installation ==
1 .Upload plugin-name.php to the /wp-content/plugins/ folder
2. Upload an mobile theme to the /wp-content/themes/' folder
3. Activate the plugin
4. Open the 'Subdomain theme'options page under the 'Appearance' menu in WordPress.
5. Enter sbudomain name, subdomain url, and select an subdomain  compatible WordPress theme
